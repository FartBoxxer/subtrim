# SubTrim Marketing Assets

Everything needed to launch and promote SubTrim without writing new copy.

## File map

| File | Purpose |
|------|---------|
| `directories.csv` | 30 ranked directories to submit to (P1/P2/P3) |
| `submission-copy.md` | Paste-ready taglines, descriptions, categories, tags |
| `show-hn.md` | Show HN post draft + comment playbook |
| `product-hunt-kit.md` | PH launch timeline, listing fields, maker comment, day-of playbook |
| `blogger-outreach.md` | Cold email template + target site list for listicle placements |
| `twitter-launch-thread.md` | 10-tweet launch thread + reply playbook |
| `screenshots/` | Generated by `npm run screenshots` |

## Recommended execution order

### Week 1 — Foundation
1. Run `npm run build && npm run preview` then `npm run screenshots` in another shell to capture all gallery images
2. Submit to P1 directories (AlternativeTo, OpenAlternative, IndieHackers, SaaSHub, Peerlist) — 2-3 hours total
3. Generate IndexNow key, deploy `public/<key>.txt`, run `npm run indexnow` to push 193 URLs to Bing

### Week 2 — Launch
4. Post Show HN (Tue-Thu, 9-11am ET)
5. Post Twitter launch thread same morning
6. Queue Product Hunt launch for following Tuesday

### Week 3 — Distribution
7. Submit blogger outreach batch (30 emails in one sitting)
8. Submit to P2 directories
9. Follow up on any replies from week 2

### Ongoing
10. New blog posts every 2 weeks targeting long-tail keywords
11. Reply to every HN/Reddit/Twitter mention within 24 hrs
12. Re-run `npm run indexnow` after each content batch

## Don't waste time on

- Instagram (wrong audience for finance tools)
- TikTok without sustained video output
- Cold DMs on LinkedIn (dead channel in 2026)
- Paying for submission services that "submit to 500 directories" (most are dead)
- Reddit automation (instant ban, permanent)

## Budget guidance

- $0 needed for first 3 weeks (all organic + Show HN + PH)
- $100-300 optional for Reddit/Google Ads once the funnel is validated
- $50 optional for Uneed or Launching Next fast-track slots if you want immediate placement
